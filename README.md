## 실습

    - MNIST
    - face detection
    - object detection
## 활용 프로젝트
  - 사람 인식 충돌 방지 로봇
      - 사용된 딥러닝 모델: Face Detection
      - 동작 설명: Face detection을 통해 얼굴을 인식하고, 가운데 좌표를 추출하여 로봇이 사람을 향하도록 회전하는 동시에 사람과의 충돌을 피하도록 함
  - 사람 따라가는 로봇
      - 사용된 딥러닝 모델: Face Detection
      - 동작 설명: Face detection을 통해 얼굴을 인식하고, 가운데 좌표를 추출하여 로봇이 따라가게 만듬
  - 라인 트레이서
      -AI적 요소 : Hough Transformation
      -로봇 : Two Wheel Robot
  - 제스처 인식 로봇 제어
